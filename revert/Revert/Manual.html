Revert

The Revert Plugin lets you reload an image from disk without losing your views.

Overview ↥

Reverting an image in Krita is currently something you must do manually: close the existing image, then re-open it. This becomes tedious if you have multiple views, perhaps in multiple windows, each carefully positioned and configured with canvas mirroring, wraparound, rotation and so on, that you then have to manually restore.

This plugin takes a snapshot of the views on your current image, closes it, reopens it, then recreates the old views.

You can find the Revert action in the File menu. You can also add a button to your toolbar from Settings → Configure Toolbars, and you can assign a shortcut from Settings → Configure Krita → Keyboard Shortcuts → Menu → File.

Dialog

A dialog appears when you press revert to confirm you wish to reload your file. It presents the option to reuse current views (the default behaviour described above) or to discard them (equivalent to closing and reopening as normal - the new image will appear in a single view in the active window in the default position).
